# IT & Cybersecurity Labs

This repository contains hands-on labs and documentation related to
IT support, system administration, and cybersecurity fundamentals.

## Labs Included
- Active Directory (User & Group Management)
- Splunk SIEM (Basic Searches & Alerts)
- Nessus Vulnerability Scanning
- SCCM / Endpoint Management (Lab Practice)

## Tools & Technologies
- Windows Server
- Active Directory
- Splunk
- Nessus
- Linux
- PowerShell (basic)

## Notes
- Labs are performed in a safe, non-production environment.
- No real user data or credentials are included.
